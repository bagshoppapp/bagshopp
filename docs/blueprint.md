# **App Name**: Blank Canvas

## Core Features:

- Basic Editor: A simple text editor that allows users to input and format text directly on the page.
- Content Saving: The application can save text data in the browser's local storage.
- Fullscreen Editing: Allows users to expand the editor to fullscreen for a distraction-free writing environment.
- Download Feature: Allows users to download content directly in plain text or rich text formats (.txt or .rtf).

## Style Guidelines:

- Primary color: Light beige (#F5F5DC) for a clean, unobtrusive background.
- Background color: Soft white (#FFFFFF) to provide a clear editing space.
- Accent color: Pale blue (#ADD8E6) for interactive elements like buttons and links.
- Font: 'Inter', sans-serif, for a modern and readable text.
- Minimalist icons for actions like saving and fullscreen mode, focused on clarity.
- Clean, full-screen layout focused on the editing space, with unobtrusive controls.
- Subtle transitions for toggling fullscreen and saving to avoid distractions.
